PROGRAMAÇÃO WEB - HTML

1 - Primeiras impressões

A linguagem HTML é uma linguagem de programação web muito utilizado para a construção de páginas web. A linguagem surgiu através da junção dos dois padrões: HyTime e SGML.

Quando falamos de HTML, falamos da forma como as páginas web estão organizadas, nomeadamente das estruturas das páginas, da definição de títulos, alinhamentos do texto e das hiperligações para outras páginas.

A estrutura do texto está organizada da seguinte forma:
  • < html > - Define o ponto de partida de uma página HTML e todos os contéudos inclúidos dentro da página são tratados como uma série de códigos HTML.

  • < head > - Define o que será o cabeçalho da página HTML, que traz consigo as informações relacionadas com o documento aberto.

  • < body > - Define o contéudos principais (o corpo) da página HTML, que será mostrado na página.

    • No < body > , pode-se atribuir vários comandos comuns ná pagina web, como por exemplo a cor de fundo, tipos de letra, divisões, títulos, entre outras.
