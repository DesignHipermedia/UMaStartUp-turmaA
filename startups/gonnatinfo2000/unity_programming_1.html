PROGRAMAÇÃO UNITY

1 - Primeiras impressões

O Unity, da Unity Tecnhologies é um programa que permite programar e criar jogos. Com o Unity, o utilizador tem as capacidades de poder criar jogos tanto em 2D como em 3D.

O Unity têm como suporte os seguintes APIs: o Direct3D, OpenGL, MacOS e Linux.P Para a criação de scripts, é usado o MonoDevelop.

O Unity tambêm tem como suporte a criação de vértices, fragmentos, tesselação e shaders.
