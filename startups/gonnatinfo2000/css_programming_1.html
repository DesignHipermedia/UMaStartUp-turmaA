PROGRAMAÇÃO WEB - CSS

1 - Primeiras impressões

A linguagem CSS (Cascade Style Sheet) é uma linguagem de programação web muito utilizado para adicionar estilos nas páginas web.

A lingaugem CSS trata-se na adição de estilos a páginas HTML, nomeadamente nas cores do fundo, espaçamentos entre textos, tipos de letra e posição de itens.

Para adicionar uma divisão, utiliza-se o comando < div >. Este comando serve para criar uma divisão no corpo da página.
