PROGRAMAÇÃO WEB - JAVASCRIPT

1 - Primeiras impressões

A linguagem JavaScript é uma linguagem de programação web que é muito utilizada para a criação e execução de scripts para páginas web interativas.

A linguagem JavaScript é considerada como uma linguagem de programação de alto nível, em que o seu tipo de dados é multi-paradigma.

O Javascript apresenta recursos fechados, executa funções de alta ordem e possui APIs que servem para trabalhar com textos, matrizes, datas, expressões, etc.

O objetivo é alterar a frase escrita em HTML para uma frase escrita em JavaScript, através de um clique no botão "Avançar".

Para associar o JavaScript a um comando HTML, é preciso criar uma identificação dentro de um comando HTML ou divisão.

Para atribuir uma identifcação, escreve-se o comando id dentro do parágrafo (< p >). E vamos dar essa identificação o nome "change".

No botão Avançar, podemos aplicar o JavaScript de forma a que, ao clicar, a frase escrita em HTML seja alterada para a frase escrita em JavaScript.

Para isso, aplicamos o comando onclick e dentro dele, o document.getElementbyId para obter o elemento através do id e escreve-se o nome da identificação "change" escrito dentro do parágrafo.

Para escrever algum texto em JavaScript, usamos o atributo .innerHTML e dentro de parentesis, escrevemos a frase "Agora a página HTML tem JavaScript."
