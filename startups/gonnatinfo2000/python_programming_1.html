PROGRAMAÇÃO PYTHON

1 - Primeiras impressões

A linguagem Python é uma linugagem de programação de alto nível e de multi-programação, com suporte a programação orientada a objetos.

O Python combina uma sintaxe clara com os recursos fortes existentes na sua biblioteca padrão e por frameworks desenvolvidos por terceiros.

Uma das principais caracterísiticas da linguagem Python é que permite efetuar a leitura fácil de códigos e exige algumas linhas de código comparando com as outras linguagens de programação.

Exemplo


Vamos então escrever 3 contas e 2 proposições num ficheiro Python.



Para a 1a conta, vamos a efetuar a soma (+) de dois números, o 2 e o 4.

Para a 2ª conta, vamos a efetuar a multiplicação (*) de dois números, o 7 e o 5.

Para a 3ª conta, vamos a efetuar o resto da divisão (%) de dois números, o 2 e o 4.

Para a 1ª proposição, vamos a verificar se a subtração (-) do 8 e do 2 é igual a 6.

Para a 2ª proposição, vamos a verificar se a divisão (/) de 24 por 4 não é igual a 6.

Para mostrar os resultados no Shell do Python, é preciso escrever o comando print().

O comando print() é utilizado para escrever qualquer texto, números, contas e proposições. Tudo o que escrito no comando será apresentado no Shell do Python.

O resultado apresentado no Shell deverá ser o seguinte:




Como se pode verificar, no Shell será mostrado os resultados das 3 contas feitas acima.

Nas duas proposições, os resultados são diferentes, isto porque essas contas servem para verificar se a conta é igual ao resultado certo:

Na 1ª proposição, o resultado é verdadeiro (True), pois a subtração de 8 por 2 é igual 6.

Na 2ª proposição, o resultado é falso (False), pois a divisão de 24 por 4 é igual a 6.
