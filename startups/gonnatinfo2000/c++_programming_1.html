PROGRAMAÇÃO C++

1 - Primeiras impressões

A lingugagem C++ é uma linguagem de programação de multi-programação utilizada para a criação de programas e aplicações de computadores.

C++ foi desenvolvido para suportar multi-programação, fornecer escolhas aos utilizadores e para ser mais compatível com o possível, apenas fornecendo transições simples para o código.

A sua biblioteca incorpora a biblioteca da linguagem C, incluindo algumas alterações para ter um melhor funcionamento com as novas funcionalidades criadas.
